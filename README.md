# MdPdf

A Markdown-to-PDF converter for Android with live preview, KaTeX math, syntax highlighting, and multiple themes.

## Features

- **Live Preview** — Editor, Split, and Preview modes
- **KaTeX Math** — Inline and display math rendering
- **Syntax Highlighting** — Via Prism.js with multiple themes
- **PDF Export** — Native A4 PDF generation with proper formatting
- **Themes** — Default, Academic, Dark, and Minimal
